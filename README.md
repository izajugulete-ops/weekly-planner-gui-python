# 🗓️ Weekly Planner & Task Manager (Python GUI)

![Python](https://img.shields.io/badge/Python-3.x-blue.svg)
![GUI](https://img.shields.io/badge/Interface-Tkinter-orange.svg)
![Storage](https://img.shields.io/badge/Data-JSON-lightgrey.svg)

A desktop productivity application designed to help users organize their weekly schedule efficiently. Featuring a color-coded interface and automated data persistence, this tool provides a seamless way to manage daily tasks and general notes.

## ✨ Features

* **Visual Weekly Layout:** Organized grid view for all seven days of the week, with distinct color-coding for easy navigation.
* **Persistent Storage:** Integrated JSON-based saving system that automatically tracks tasks, completion states, and custom colors across sessions.
* **Task Customization:** Right-click context menus allow users to set priority levels (High, Medium, Low) using color indicators.
* **Auto-Save Functionality:** Real-time data synchronization that triggers on text changes, checkbox toggles, or focus loss.
* **Integrated Notes Section:** A dedicated global notes area for long-form thoughts or general reminders.
* **Interactive UI:** Dynamic task management allowing users to add, individual delete, or clear entire daily lists with a single click.

## 🛠️ Tech Stack

* **Language:** Python 3.x
* **GUI Framework:** `Tkinter` & `ttk` for a responsive desktop experience.
* **Data Management:** `json` module for structured data serialization.
* **File I/O:** `pathlib` for robust cross-platform file path handling.
